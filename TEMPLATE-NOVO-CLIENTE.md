## 5. Checklist de Implementação

### Coletar
- [ ] Briefing do negócio preenchido
- [ ] Tom de voz definido
- [ ] Referências de conteúdo (perfis, estilo)
- [ ] Acesso às ferramentas (se necessário)
- [ ] Material existente (copies, páginas, aulas)

### Criar
- [ ] Repositório GitHub (privado)
- [ ] Estrutura do cérebro (pastas + contexto)
- [ ] Arquivos dos agentes (SOUL, AGENTS, IDENTITY, TOOLS)
- [ ] Skills e rotinas configuradas

### Conectar
- [ ] API Key da IA (OpenAI ou Anthropic — cliente providencia)
- [ ] Servidor (Hostinger/Hetzner — cliente providencia)
- [ ] Canal de comunicação (Telegram / Discord / WhatsApp)
- [ ] Testar se funciona

### Entregar
- [ ] Agentes no ar com conhecimento mínimo
- [ ] Treinamento inicial com o cliente
- [ ] Documentação entregue

### Evoluir
- [ ] Cliente usando e dando feedback
- [ ] Agentes aprendendo com cada interação
- [ ] Suporte pós-entrega (30 dias se Squad / 12 meses se High Ticket)
