# Tactical Empathy (Voss) — Scripts Canônicos

**Tipo:** metodologia (scripts práticos)
**Pré-requisito:** ler `tactical-empathy-voss-principios.md` e `tactical-empathy-voss-execucao.md`.

---

## Como usar este documento

Voss não é metodologia conversacional sequencial como SPIN. **São ferramentas pontuais** que vendedor invoca quando há tensão emocional, resistência, conflito.

Cada script vem com:
- **Ferramenta Voss** (Mirroring, Labeling, Calibrated Question, etc)
- **Quando usar**
- **Adaptação Pinguim**

---

## 1. Mirroring (Espelho) — repetir 1-3 últimas palavras

### Script M.1 — Espelho clássico
```
Cliente: "Eu acho que esse programa é muito caro pra mim."
Vendedor: "Muito caro pra você?"
[silêncio]
```

**O que acontece:** cliente preenche silêncio com mais informação. Frequentemente revela razão real.

### Script M.2 — Espelho amigável
```
Cliente: "Tô em dúvida se vale a pena."
Vendedor: "Vale a pena?"
[espera]
```

**Princípio Voss:** mirror é a forma menos invasiva de fazer cliente continuar falando. Cliente que ouve a própria frase espelhada elabora mais.

**Anti-padrão:** mirroring virando papagaio. Use 2-3 vezes na call, não 10.

---

## 2. Labeling (Rotular emoção)

### Script L.1 — Label da preocupação
```
"Parece que você tá com alguma preocupação sobre [X]."
```

**Quando usar:** percebe que cliente está com algo mas não verbaliza.

### Script L.2 — Label da hesitação
```
"Sinto que você tá hesitando aí."
```

**Princípio:** quando você nomeia a emoção do cliente, ele relaxa. Sente que foi visto.

### Script L.3 — Label da frustração
```
"Parece que essa experiência passada com [X] te marcou bastante."
```

**Adaptação Pinguim:** "Parece que algo no curso anterior te queimou."

### Script L.4 — Label do desejo
```
"Parece que isso aí é importante pra você."
```

**Quando usar:** confirma valor antes de propor.

**Princípio Voss:** sempre comece com "parece" ou "sinto que" — nunca "você está". Afirmar emoção do outro gera resistência. Sugerir abre conversa.

**Anti-padrão:** "como você se sente?" — Voss bane essa pergunta. Soa terapêutico, destrói credibilidade comercial. **Use labeling**, não "como se sente".

---

## 3. Calibrated Questions (Perguntas calibradas)

**Regra:** começam com "**como**" ou "**o quê**". **NUNCA** com "por quê" (soa acusatório).

### Script CQ.1 — Como funciona
```
"Como você decide nessas situações?"
"Como isso funciona pra você hoje?"
```

### Script CQ.2 — O que acontece
```
"O que aconteceria se você não fizesse nada?"
"O que vai mudar se você decidir agora?"
```

### Script CQ.3 — A pergunta canônica de Voss
```
"O que precisa estar verdade pra você decidir?"
```

**Princípio:** força cliente a verbalizar critérios. Frequentemente revela objeção real.

### Script CQ.4 — O que aconteceu (recuperação)
```
"O que aconteceu na sua experiência anterior que faz você ter cuidado agora?"
```

**Quando usar:** com cliente queimado por curso/serviço passado.

### Script CQ.5 — Como você gostaria
```
"Como você gostaria que isso fosse?"
"Como seria o cenário ideal pra você?"
```

**Princípio:** cliente que verbaliza cenário ideal entrega seus critérios — você só precisa entregar isso.

## 4. The "No"-oriented question

### Script NO.1 — Permissão pra continuar
```
"Você acha um péssimo momento pra falar?"
```

**Por que funciona:** cliente diz "não, tudo bem" — e isso é um SIM produtivo. Voss observa que humanos têm aversão a "sim" mas conforto em "não".

### Script NO.2 — Confirmar crítica
```
"Seria absurdo se eu te ligasse na próxima quarta?"
```

**Cliente responde:** "não, não seria absurdo" → você marcou call sem pressionar.

### Script NO.3 — Validar interesse
```
"Você desistiu de [resolver problema X]?"
```

**Por que funciona:** cliente diz "não, não desisti" — confirma que ainda quer. Vendedor ganha confiança pra continuar conversa.

**Princípio:** "Sim" cria comprometimento (e gera resistência). "Não" cria conforto (e abre conversa). Use perguntas que permitam "não" produtivo.

## 5. Voz de DJ noturno (Late-Night DJ Voice)

Voss prega 3 tons de voz:
- **Tom assertivo** (poucos usos — quase só pra autoridade pontual)
- **Tom direto/positivo** (a maioria das interações)
- **Tom DJ noturno** (lento, profundo, calmo) — para desarmar emoção

### Quando usar tom DJ noturno
- Cliente muito agitado/frustrado
- Negociação travada com tensão
- Pedido de reembolso

### Como soa
**Errado (tom direto):** "Espera, deixa eu te explicar." [rápido]
**Certo (DJ noturno):** "Entendo o que você tá passando." [pausado, voz mais grave]

**Princípio:** vendedor que combina tom calmo com labeling **desarma** cliente em conflito. Cliente percebe que está sendo ouvido, baixa guarda.

---

## 6. The Accusation Audit (Auditoria de Acusação)

**Conceito:** liste TODAS as objeções negativas que cliente pode ter contra você ANTES dele falar.

### Script AA.1 — Antes de pitch
```
"Você provavelmente está pensando: 'esse cara tá tentando me vender mais um curso caro que vai prometer e não entregar'. Talvez também 'já tentei coisa parecida antes'. E talvez 'não tenho tempo pra mais um programa que vou abandonar no meio'."

[espera reação]
```

**Princípio:** quando você lista as críticas antes do cliente, ele **não tem o que dizer**. Você desarmou a defesa antes do ataque.

### Script AA.2 — Em recuperação
```
"Eu sei que você deve estar pensando que eu vou empurrar uma venda. Talvez também esteja achando que eu sou só mais um vendedor com script. E talvez quer me mandar embora educadamente."
```

**Quando usar:** primeira mensagem de recuperação de lead frio.

**Princípio Voss:** cliente que vê suas **próprias** objeções verbalizadas pelo vendedor sente que foi entendido. **Defesa cai sozinha.**

## 7. The "How am I supposed to do that?" (Como devo fazer isso?)

### Script H.1 — Resposta a pedido absurdo
```
Cliente: "Quero 50% de desconto."
Vendedor: "Como eu vou conseguir entregar isso?"
[espera]
```

**Por que funciona:**
- Não é "não" direto (que gera conflito)
- Cliente fica responsável por pensar na solução
- Frequentemente cliente recua sozinho

### Script H.2 — Em negociação difícil
```
Cliente: "Preciso começar amanhã, sem pagar antes."
Vendedor: "Como eu vou explicar isso pro meu time? Não tenho como abrir exceção dessa forma."
```

**Princípio:** desloca o peso da decisão pro cliente. Em vez de "não", você apresenta limitação real e pede solução.

---

## 8. Bending Reality (Curvar Realidade)

**Quando usar:** cliente apresenta deadline ou limite que parece inegociável. Em vez de combater diretamente, **curvar** a percepção.

### Script BR.1 — Reframe de deadline
```
Cliente: "Preciso fechar até sexta ou desisti."
Vendedor: "Entendo a pressa. Pra a gente fechar até sexta com qualidade, o que precisa acontecer hoje?"
```

**Princípio:** aceita o deadline mas faz cliente listar o que precisa fazer pra cumprir. Frequentemente cliente percebe que é viável fazer com mais calma.

### Script BR.2 — Reframe de orçamento
```
Cliente: "Tenho R$ 5K de orçamento, programa custa R$ 7K."
Vendedor: "OK, R$ 5K é o orçamento. Como conseguiríamos chegar em R$ 7K com algum recurso adicional? Parcelamento, bônus, capital de giro do negócio?"
```

**Princípio:** aceita o "não posso pagar X" mas explora caminhos alternativos sem pressionar.

## 9. Sequência canônica de Voss (call de recuperação tensa)

```
1. Tom DJ noturno + Label: "Parece que você ficou frustrado com [X]."
2. Mirroring: "Frustrado com [X]?"
3. [cliente elabora]
4. Calibrated question: "O que aconteceu de fato?"
5. [cliente conta histórico]
6. Accusation Audit: "Você provavelmente acha que eu vou empurrar mais uma venda."
7. Calibrated question: "O que precisa estar verdade pra você reconsiderar?"
8. [cliente verbaliza critérios]
9. Negociação calibrada baseada nos critérios
```

## Princípios universais Voss

### 1. Tom > Conteúdo
*Como* você diz pesa mais que *o que* você diz. Voz calma + labeling resolve mais que argumento.

### 2. "Não" é amigo
Cliente que pode dizer "não" produtivo se sente confortável. Cliente forçado a "sim" se defende.

### 3. Empatia ≠ concordar
Você pode validar emoção sem aceitar pedido. "Entendo sua frustração" não significa "vou te dar desconto".

### 4. Pergunta com "como"/"o que" > "por quê"
"Por quê" soa acusação. "Como"/"o que" soa colaboração.

### 5. Silêncio depois de mirror/label
Vendedor preenche silêncio = perde insight. Cliente preenche silêncio = entrega informação.

### 6. Auditoria de acusação desarma defesa
Liste a crítica do cliente antes dele. Defesa cai sem briga.

---

## Como o agente comercial Pinguim usa Voss

**Em situações específicas:**

- **Cliente hostil em recuperação:** Sequência de 9 passos
- **Pedido de reembolso:** Tom DJ + Label + Calibrated question
- **Negociação de preço travada:** "How am I supposed to do that?" + Bending Reality
- **Lead defensivo no primeiro contato:** Accusation Audit
- **Cliente emocional em qualquer momento:** Mirroring + Labeling

**Voss não substitui SPIN/MEDDIC** — complementa quando há tensão. Vendedor maduro usa SPIN como esqueleto e Voss como tempero quando emoção aparece.
