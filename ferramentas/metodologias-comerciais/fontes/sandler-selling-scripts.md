# Sandler Selling — Scripts Canônicos

**Tipo:** metodologia (scripts práticos)
**Pré-requisito:** ler `sandler-selling-principios.md` e `sandler-selling-execucao.md`.

---

## Como usar este documento

Sandler tem **3 mecânicas distintas** que viraram marca registrada da metodologia:

1. **Up-front Contracts** — combinado de início de call
2. **Pain Funnel** — sequência de 8 perguntas pra escavar dor
3. **Reverse / Negative Reverse Selling** — devolver pergunta pro cliente

Esses 3 são o que diferenciam Sandler de SPIN/Challenger/MEDDIC.

---

## 1. Up-front Contract (Combinado de início de call)

**Conceito:** estabelecer regras claras da call no início — tempo, pauta, possíveis resultados, próximos passos.

**Por que existe:** evita "consultoria grátis" (cliente extrai informação sem fechar) e cria contrato psicológico.

### Script U.1 — Up-front clássico
```
"Antes da gente entrar nas perguntas, queria combinar 4 coisas:

1. **Tempo:** temos 30 minutos. OK pra você?

2. **Pauta:** vou te fazer perguntas pra entender seu cenário. Depois, se fizer sentido, te apresento como a gente trabalha. Senão, te dou direção mesmo sem fechar nada.

3. **Possíveis resultados:** ao final dessa call, vão ser 3 cenários possíveis: (a) faz sentido seguir, agendamos próxima conversa, (b) não faz sentido, encerramos amistosamente, (c) precisa de tempo, marcamos retorno. Os 3 são OK pra mim.

4. **Próximo passo:** ao final, eu pergunto onde você está, sem rodeios. Pode ser sim, pode ser não, pode ser 'preciso pensar'. Aceito qualquer resposta.

Combinado?"
```

**Princípio:** depois desse contrato, cliente que pular pra "vou pensar" no fim sem clareza pode ser confrontado: "combinamos no início que você daria resposta direta. Tá em qual cenário?"

### Script U.2 — Up-front simplificado (call rápida)
```
"Vou te fazer algumas perguntas. Se em 15 minutos a gente concluir que não faz sentido, sem stress — encerramos. Se fizer, conversamos sobre próximo passo. OK?"
```

## 2. Pain Funnel (8 Perguntas em Sequência)

**Conceito:** sequência específica pra escavar dor do cliente, do superficial ao emocional profundo.

### Pain Funnel canônico

```
1. "Me conta um pouco mais sobre isso..."
   [cliente descreve problema superficialmente]

2. "Pode dar um exemplo específico?"
   [cliente concretiza]

3. "Há quanto tempo isso é um problema?"
   [cliente verbaliza duração]

4. "O que você já tentou pra resolver?"
   [cliente revela cicatrizes]

5. "Funcionou?"
   [cliente reconhece tentativas frustradas]

6. "Quanto isso te custou? (em tempo, dinheiro, oportunidade)"
   [cliente quantifica]

7. "Como você se sente sobre isso?"
   [cliente verbaliza emoção]

8. "Você desistiu?"
   [força "não, não desisti" — confirma que ainda quer resolver]
```

**Princípio:** cada pergunta aprofunda. Pular pergunta = dor superficial. Aplicar todas = cliente sai **convencido pela própria boca** que precisa resolver.

### Aplicação Pinguim

```
Cliente: "Acho que minhas vendas tão fracas."

1. "Me conta mais sobre isso. Como você sabe que tão fracas?"
2. "Pode dar um exemplo? Tipo, do último mês."
3. "Há quanto tempo tá assim?"
4. "O que você já tentou pra reverter?"
5. "Funcionou?"
6. "Quanto isso te custou em receita perdida? E em tempo?"
7. "Como isso afeta você no dia a dia? Sua família, seu humor?"
8. "Você desistiu?"
```

**Resultado típico:** cliente que começou dizendo "vendas fracas" termina admitindo dor de R$ 100K perdidos, queimação emocional, mas ainda não desistiu. **Vendedor ganhou contexto pra fechar.**

## 3. Reverse / Negative Reverse Selling

**Conceito:** devolver pergunta pro cliente em vez de responder direto. Inverte dinâmica.

### Script R.1 — Reverse simples
```
Cliente: "Quanto custa?"
Vendedor: "Posso te perguntar antes — qual orçamento você tinha imaginado?"
```

### Script R.2 — Reverse pra qualificar
```
Cliente: "Vocês trabalham com [nicho específico]?"
Vendedor: "Trabalhamos. Pergunto: por que isso é importante pra você?"
```

### Script R.3 — Negative Reverse Selling (reverse com dúvida)
```
Cliente: "Acho que esse curso é o que preciso."
Vendedor: "Talvez seja, mas talvez não seja. Posso entender melhor seu cenário antes de afirmar?"
```

**Princípio Sandler:** cliente que ouve "talvez não seja" se sente menos pressionado, **fecha mais**. Inverte expectativa de "vendedor empurrando" pra "vendedor avaliando se vale".

### Script R.4 — Negative Reverse em proposta
```
Cliente: "Vou ver se faz sentido pra mim."
Vendedor: "Honestamente, em 50% dos casos como o seu, não faz sentido. Por isso pergunto: pelo que você me contou, você está mais perto do 'faz sentido' ou do 'não faz'?"
```

**Por que funciona:** cria urgência reversa. Cliente que estava confortável adiando agora **defende** o "faz sentido" pra não cair no 50% que não fechou.

## 4. Outros padrões Sandler

### Script S.1 — "Help me understand"
```
"Me ajuda a entender uma coisa: você falou que [X], mas também [Y]. Como esses dois se conectam?"
```

**Quando usar:** cliente fala coisas contraditórias. Forçar coerência com ferramenta suave.

### Script S.2 — "Going for the No"
```
"Olha, tudo bem se você decidir não fechar. Sério. Prefiro um 'não' agora do que um 'vou pensar' que vira nada. O que você decide?"
```

**Por que funciona:** cliente sente liberdade pra recusar. Frequentemente, ao perceber que pode dizer "não", percebe que quer dizer "sim".

### Script S.3 — "Mutual Mystification"
```
"Olha, posso ser honesto? Não sei se a gente é a melhor solução pra você. Vamos descobrir juntos?"
```

**Princípio:** humildade radical. Cliente sente parceria, não venda forçada.

## 5. Sequência canônica Sandler (call de descoberta)

```
1. Bonding (5min) — rapport breve
2. Up-front contract — combinar regras
3. Pain Funnel — 8 perguntas pra escavar dor
4. Budget — descobrir capacidade de pagamento
5. Decision — descobrir processo decisório
6. Apresentação (só se passou pelas 5 anteriores)
7. Going for the No — pedir resposta direta
```

## Princípios universais Sandler

### 1. "Pra fechar uma porta, abre outra"
Sandler insiste: melhor recusar deal mau do que carregar deal eterno. "Vou pensar" indefinido = "não" disfarçado.

### 2. Help the prospect win or lose fast
Cliente decide rápido (sim ou não), vendedor é grato pelos dois. Ambos ganham tempo.

### 3. People buy emotionally, justify logically
Pain Funnel chega na emoção (perguntas 7-8). É lá que decisão acontece.

### 4. No sales call without an up-front contract
Toda call séria começa com combinado claro. Sem isso, vendedor é "abusado".

### 5. The professional rejects rejection
Vendedor Sandler não leva "não" pessoalmente. "Não" é sinal pra ir pro próximo deal.

## Como agente comercial Pinguim usa Sandler

**Cenários onde Sandler brilha:**
- Mercado com muito tire-kicker (curioso sem intenção)
- Vendedor sente que está sendo "consultado" sem fechamento
- Cliente procrastinador clássico ("vou pensar" eterno)
- Cultura corporativa formal (Sandler tem estrutura previsível)

**Cenários onde NÃO usa Sandler:**
- Cliente já comprado (use Hormozi/Cialdini)
- Cliente emocional/em conflito (use Voss)
- Low ticket digital (use low-ticket-digital)

**Combinação útil:**
- Sandler como esqueleto do ciclo
- SPIN dentro do Pain Funnel pra perguntas mais sofisticadas
- Voss em momentos de tensão emocional
- MEDDIC pra qualificação paralela
