# TOOLS.md — Suporte Mentorados

## Ferramentas essenciais (Fase 4 — MVP)

### 1. WhatsApp API (via Sendflow)
- **Pra que serve:** Canal principal de atendimento
- **Credenciais:** Sendflow
- **Custo:** Sendflow plano pago

### 2. Telegram Bot API
- **Pra que serve:** Canal alternativo
- **Custo:** Gratis

### 3. Acesso ao cerebro (leitura)
- **Pra que serve:** Material de Lira, Taurus, Orion + FAQ + calls gravadas
- **Integracao:** Filesystem direto

### 4. Hotmart API (leitura)
- **Pra que serve:** Confirmar se o aluno tem acesso ativo a mentoria
- **Custo:** Gratis

### 5. OpenAI API
- **Custo:** Pay-per-use (resposta consultiva precisa qualidade)

## Ferramentas ideais (Fase 5+)

### 6. Sistema de tickets
- **Pra que serve:** Escalar pro mentor humano em decisoes estrategicas
- **Integracao:** Email ou Discord dedicado

### 7. Transcricao de calls (Whisper)
- **Pra que serve:** Expandir base de conhecimento a medida que calls vao acontecendo
- **Custo:** OpenAI Whisper (barato)
