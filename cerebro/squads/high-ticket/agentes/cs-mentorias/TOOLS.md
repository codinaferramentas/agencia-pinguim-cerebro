# TOOLS.md — CS Mentorias

## Ferramentas essenciais (Fase 4 — MVP)

### 1. WhatsApp API (via Sendflow)
- **Pra que serve:** Canal principal com mentorados (ticket alto = expectativa de atencao rapida)
- **Credenciais:** Sendflow
- **Custo:** Sendflow plano pago

### 2. Hotmart API (leitura)
- **Pra que serve:** Dados dos mentorados fechados, acesso ao programa, data de fechamento
- **Custo:** Gratis

### 3. Acesso ao cerebro
- **Pra que serve:** Jornada de cada mentoria + historico de cada mentorado
- **Integracao:** Filesystem direto

### 4. Discord API
- **Pra que serve:** Escalar sinais de churn pro Estrategista HT e mentor humano
- **Custo:** Gratis

### 5. OpenAI API
- **Custo:** Pay-per-use

## Ferramentas ideais (Fase 5+)

### 6. Google Calendar API
- **Pra que serve:** Agendar checkpoints (quinzenal Lira, mensal Taurus, sob demanda Orion)
- **Custo:** Gratis

### 7. Sistema de NPS/survey
- **Pra que serve:** Coletar satisfacao ao longo da jornada
- **A decidir:** Typeform? Tally? Integracao nativa?

### 8. Banco de cases
- **Pra que serve:** Registrar depoimentos e resultados pra alimentar Copy HT
