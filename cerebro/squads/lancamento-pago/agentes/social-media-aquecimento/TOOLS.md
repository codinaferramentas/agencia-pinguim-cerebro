# TOOLS.md — Social Media de Aquecimento

## Ferramentas essenciais (Fase 4 — MVP)

### 1. Acesso ao cerebro (leitura)
- **Pra que serve:** Briefing do Estrategista + historico de posts que performaram
- **Integracao:** Filesystem direto

### 2. Sistema de acionamento Story Chief + Copy Chief
- **Pra que serve:** Consultar Chiefs pra roteiros de Reel e legendas persuasivas
- **Integracao:** Mensageria interna

### 3. OpenAI API
- **Custo:** Pay-per-use

## Ferramentas ideais (Fase 5+)

### 4. Instagram Graph API (leitura)
- **Pra que serve:** Ver engajamento dos posts, identificar horarios que funcionam
- **Integracao:** Meta Business Manager
- **Credenciais:** Long-lived token do perfil do expert
- **Custo:** Gratis

### 5. Buffer/Hootsuite API (agendamento)
- **Pra que serve:** Agendar posts automaticamente
- **A decidir:** Usar ferramenta externa ou deixar equipe postar manualmente?

### 6. Tendencias do nicho (SerpAPI ou similar)
- **Pra que serve:** Identificar trending pra aproveitar em posts
- **Custo:** SerpAPI tem plano gratuito pequeno

## Observacoes

- Agente trabalha com **janela curta** (14 dias pre-desafio)
- Entrega plano editorial — **humano publica** (plataformas de agendamento custam)
- Pode ser modelo mini do LLM (tarefa menos critica que copy de pagina)
