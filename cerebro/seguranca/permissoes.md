# Permissionamento — Agência Pinguim

## 1. Acesso ao cérebro por agente

| Agente | Leitura | Escrita |
|--------|---------|---------|
| **Chief (Pinguim)** | Cérebro inteiro | Cérebro inteiro |
| **Agentes pessoais** | Cérebro inteiro | Cérebro inteiro |
| **Estrategistas** | Cérebro inteiro + pode acionar especialistas | empresa/ + área do produto |
| **Agentes de área** | empresa/ + sua área | empresa/ + sua área |
| **Agentes de marketing** | empresa/ + areas/marketing/ | empresa/ + areas/marketing/ |
| **YouTube** | empresa/ + areas/youtube/ | empresa/ + areas/youtube/ |
| **Suporte (roteador)** | empresa/ + areas/cs/ | areas/cs/ |
| **Suporte (especialistas)** | empresa/ + contexto da mentoria | contexto da mentoria |

## 2. Regras gerais

1. Agente de área nunca acessa dados de outra área
2. Chief (Pinguim) é o único com acesso irrestrito além dos pessoais
3. Ações externas (enviar mensagem, publicar, gastar dinheiro) sempre pedem confirmação humana
4. Suporte ao aluno tem escopo mínimo — só o necessário pra responder dúvidas

## 3. Canais

| Canal | Agente(s) que escuta | Quem pode acionar |
|-------|---------------------|-------------------|
| Discord (interno) | Chief (Pinguim) + agentes de área | Equipe inteira |
| [Canal pessoal — a definir] | Agente pessoal de cada sócio | Só o sócio |
