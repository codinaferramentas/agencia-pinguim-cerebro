# 🦞 Projeto Squad — Agência Pinguim

## Visão Geral

Projeto de implementação de um **Squad de Agentes de IA** para a Agência Pinguim, com o objetivo de criar uma operação mais enxuta, automatizada e escalável.

### Objetivos

1. **Criar um squad de agentes de IA** que trabalhem junto com o time humano da Agência Pinguim
2. **Reduzir custos operacionais** com automação de processos internos
3. **Padronizar e otimizar processos** — mitigar erros, acelerar entregas
4. **Criar um produto replicável** — case de sucesso que pode ser vendido para alunos e mentorados
5. **Documentar tudo** — para que qualquer agência possa replicar a estrutura

### Contexto do Negócio

A Agência Pinguim atua com **cursos e mentorias de marketing**. O projeto nasce da necessidade de:
- Ter agentes de IA trabalhando internamente (relatórios, atendimento, processos)
- Reduzir custo de staff mantendo ou aumentando a qualidade
- Criar um modelo replicável que possa ser oferecido como produto/serviço aos mentorados

### Tecnologia Base

- **OpenClaw** — Plataforma open-source de agentes de IA autônomos
- **GitHub**: [openclaw/openclaw](https://github.com/openclaw/openclaw)
