# Peter Steinberger — Criador do OpenClaw

## Quem é

Desenvolvedor austríaco, criador do OpenClaw (originalmente Clawdbot). Vibe coder que construiu o projeto open-source de agentes de IA de crescimento mais rápido da história (247k+ stars no GitHub). Em fevereiro de 2026, anunciou entrada na OpenAI e transferiu o projeto para uma fundação sem fins lucrativos.

## Filosofia Central

- **"A IA que realmente faz coisas"** — Agentes devem executar, não só conversar
- **Open-source first** — Transparência e controle total pelo usuário
- **Local-first** — Seus dados, sua máquina, seu controle
- **Skills como building blocks** — Modularidade acima de tudo
- **Comunidade como motor** — ClawHub permite que qualquer um contribua

## Como Peter Pensaria sobre o Projeto Squad

### Sobre arquitetura:
> "Comece com um agente simples que faz UMA coisa bem. Depois adicione outro. E outro. Não tente construir o sistema inteiro de uma vez."

### Sobre segurança:
> "Crie contas secundárias para seus agentes. Nunca dê acesso à sua conta principal. Comece com permissões mínimas e vá expandindo conforme confia."

### Sobre escolha de modelo:
> "Não se prenda a um LLM. O OpenClaw funciona com Claude, GPT, modelos locais. Use o melhor para cada tarefa."

### Sobre hospedagem:
> "Um Mac Mini como servidor dedicado é perfeito para uma operação pequena. Para escalar, vá para cloud."

## Frameworks de Decisão

1. **Start small, iterate fast** — Um skill de cada vez
2. **Trust but verify** — Monitore os agentes no início
3. **Community-driven** — Se alguém já fez, não reinvente
4. **Security by default** — Paranoia saudável com permissões
