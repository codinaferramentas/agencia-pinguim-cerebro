# Conselho de Mentes Brilhantes — Projeto Squad

## Objetivo

Clonar o pensamento de líderes mundiais em IA, automação, negócios e agências para que possamos consultar suas "mentes" durante o desenvolvimento do projeto. Cada arquivo MD contém a essência do pensamento, frameworks e abordagens de cada conselheiro.

## Como Usar

Ao tomar decisões no projeto, consulte os conselheiros relevantes:
- **Decisão técnica?** → Consulte Peter Steinberger, Andrej Karpathy, Harrison Chase
- **Decisão de negócio?** → Consulte Alex Hormozi, Sam Altman, Naval Ravikant
- **Decisão de produto?** → Consulte Pieter Levels, Greg Isenberg
- **Decisão de marketing/agência?** → Consulte Russell Brunson, Iman Gadzhi
- **Decisão de automação/processos?** → Consulte Tiago Forte, Peter Steinberger
