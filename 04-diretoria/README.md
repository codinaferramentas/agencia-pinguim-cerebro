# Diretoria — Agência Pinguim

## Objetivo

Documentar o perfil, visão, estilo de decisão e prioridades de cada sócio para que os agentes de IA possam atender suas necessidades de forma personalizada.

> **IMPORTANTE**: Estes perfis precisam ser preenchidos/validados pelos próprios sócios para garantir fidelidade.

## Sócios

- [Pedro Aredes](pedro-aredes.md)
- [Michel Menezes](michel-menezes.md)
- [Luiz Costa](luiz-costa.md)

## Estrutura de Cada Perfil

Cada arquivo de sócio contém:
1. **Papel na empresa** — responsabilidades e área de atuação
2. **Estilo de gestão** — como toma decisões, prioridades
3. **Visão para o projeto** — o que espera do Squad de agentes
4. **Demandas recorrentes** — tarefas que mais consome tempo
5. **Preferências de comunicação** — como prefere receber informações
6. **Métricas importantes** — o que monitora/importa para ele

## Como os Agentes Usarão Isso

Os agentes consultarão estes perfis para:
- Adaptar tom e profundidade de relatórios por sócio
- Priorizar informações conforme o que cada um monitora
- Antecipar necessidades baseado em padrões
- Tomar decisões alinhadas com a visão de cada um
